# 农产品信息展示与售卖管理系统（乡村助农实战项目）

> 课程实训 · 乡村助农类特色实战项目

## 项目简介

面向乡村助农场景的农产品信息展示与售卖管理系统，帮助农户展示产地与产品信息，帮助消费者便捷选购，并为管理者提供订单与销量的运营数据。

## 主要功能

- 农产品上架：录入商品名称、价格、库存、图片、产地等基础信息
- 分类展示：按品类（果蔬/粮油/畜禽/特产等）筛选与展示
- 订单管理：下单、订单状态流转（待付款/待发货/已完成/已取消）
- 销量统计：按商品/品类/时间维度统计销量与销售额
- 产地信息公示：展示产地溯源、农户与种植/养殖信息
